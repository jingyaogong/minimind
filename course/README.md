# MiniMind Source Course

This directory contains a personal source-code-driven learning course built on top of MiniMind.

Original project:

- Repository: https://github.com/jingyaogong/minimind
- License: Apache-2.0

What this directory adds:

- `00_syllabus.md`: course map and learning route.
- `01_*.md` to `28_*.md`: source-guided lessons.
- `labs/`: small runnable experiments for tracing tokens, labels, shapes, losses, logprobs, rewards, and deployment surfaces.
- `impl/`: teaching implementation of selected MiniMind core ideas.
- `notes/`: personal notes and recurring mistakes.
- `portfolio/`: progress, experiment logs, implementation notes, and resume draft.

Scope:

The original MiniMind source code remains authored by the MiniMind project. This course is an added learning layer for reading, tracing, and re-implementing selected concepts from the source.
